Login Screen w/ Error

If the user encounters an error while trying to log in, they will end up on this screen. Similar to the original login screen, you will need to provide the user with 2 fields, a username field and a password field, as well as a standardized Android login button that will enable the user to login. You must also include announcement informing the user of the error, as a well offering a potential solution.

Also similar to the login screen: at the bottom of the page there should also be 2 links: One for "Forgot Your Password?" and also one for "Forgot your Username?" Unlike the login screen, we do not include the "Not a Member? Sign Up Now!" link, because if a user has reached this page, it is because they have already tried to login, and thus already have a membership.
