Boarding Pass

This is the final screen in the user’s workflow. The top half of the screen is comprised of the short code that can be used when scanning the digital bordering pass.

Beneath it is a container with information of the current passenger that the boarding pass represents, as well as information that the user would need when approaching their get to board. The first row should include the passenger’s name. The second row should display the flight number, the boarding gate, the passenger’s seat number, and the boarding zone. Finally the third row should display the flight’s estimated boarding and departing times.

Near the bottom center of the page are filmstrip pagination “bubbles” indicating to the user that there are multiple boarding passes available under this selection.

Select the "Home" button on the Android faceplate to return to the start screen.
